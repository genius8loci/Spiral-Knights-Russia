## 📖 О котлете

Spiral Knights — прекрасная кооперативная Action-RPG, но официальной русской локализации у неё нет. Этот проект исправляет ситуацию: полный перевод интерфейса, предметов, заданий, диалогов и всех игровых текстов на русский язык.

### Что переведено

| Категория | Файлы | Описание |
|-----------|-------|----------|
| 🎮 Интерфейс | `global`, `logon`, `status`, `system` | Кнопки, меню, системные сообщения |
| ⚔️ Предметы | `item`, `item-names` | Названия и описания всего снаряжения |
| 📜 Задания | `mission`, `mission-names` | Квесты и сюжетные задания |
| 💬 Диалоги | `conversation`, `prologue` | NPC, пролог |
| 🏰 Подземелья | `dungeon`, `dungeon-names` | Уровни, этажи, локации |
| 🛒 Торговля | `shop`, `trade`, `auction`, `exchange` | Магазины, аукцион, торговля |
| ⚒️ Крафт | `craft`, `design` | Ковка и рецепты |
| 👥 Социальное | `social`, `guild`, `chat`, `invite` | Гильдии, чат, друзья |
| 🐾 Существа | `monster`, `sprites` | Монстры и спрайты |
| 🏟️ PvP/Арена | `pvp`, `arena`, `lottery` | Lockdown, Blast Network |
| 📋 Прочее | `help`, `helptips`, `support`, `steam`... | Справка, подсказки, Steam |

**Итого: 41 файл, 11 600+ переведённых строк** — покрывает практически весь текстовый контент игры.
> **Файлы, которые не переведены:**
> - Spiral Knights\getdown-pro.jar\com\threerings\getdown\messages.properties
> - Spiral Knights\code\projectx-config.jar\rsrc\i18n\story.properties

### Как это работает

```
┌──────────────────────────────────────────────────────┐
│                  Запуск игры                         │
│                                                      │
│  launch-ru-steam.bat                                 │
│       │                                              │
│       ├─ Читает getdown.txt (настройки игры)         │
│       ├─ Собирает classpath и JVM-аргументы          │
│       ├─ Добавляет ru-locale.jar ПЕРВЫМ в classpath  │
│       └─ Запускает игру стандартным Java-процессом   │
│                                                      │
│  При загрузке ресурсов:                              │
│       Java ищет файлы по classpath слева направо     │
│       → ru-locale.jar (русский) найден первым        │
│       → Оригинал в projectx-config.jar пропускается  │
│                                                      │
│  Результат: русский текст вместо английского         │
└──────────────────────────────────────────────────────┘
```

### Ключевые гарантии

| Аспект | Гарантия |
|--------|----------|
| **Оригинальные файлы** | Не изменяются, не перезаписываются, не патчатся |
| **Механизм** | Стандартный приоритет Java classpath — первый JAR в пути имеет приоритет |
| **Что затрагивается** | Только текстовые `.properties` файлы из `rsrc/i18n/` |
| **Игровой код** | Не затронут — не изменяется ни один `.class` файл |
| **Сетевая игра** | Перевод локален — другие игроки видят свой язык |
| **Обновления** | Автоматический лаунчер обновляет игру как обычно |
| **Откат** | Удалите `ru-locale.jar` и запуск-скрипт — игра полностью как была |
| **Античит** | Не срабатывает — файлы игры не изменены, DLL не инжектируются |

Это аналог ресурспаков в Minecraft или языковых модов в других Java-играх — стандартная практика локализации через classpath.

## 🛠️ Инструменты проекта

Проект включает набор скриптов для работы с переводом:

| Скрипт | Назначение |
|--------|-----------|
| `build-ru-locale.bat` | Собрать `ru-locale.jar` из файлов перевода |
| `extract-originals.bat` | Извлечь английские оригиналы из игры |
| `diff-update.bat` | Найти непереведённые и проблемные ключи |
| `pack-release.bat` | Упаковать релиз для распространения |
| `launch-ru-steam.bat` | Запуск игры с переводом (Steam-версия) |
| `launch-ru-debug.bat` | Запуск с консолью для отладки |

Каждый `.bat` файл — обёртка над одноимённым `.ps1` скриптом (PowerShell).

## ❓ Чаво

**Меня могут забанить за использование перевода?**
- Нет. Перевод не модифицирует файлы игры, не внедряет код, не влияет на игровой процесс. Это просто альтернативный текст интерфейса, загружаемый через стандартный механизм Java.

**Перевод сломается после обновления игры?**
- Возможно, если разработчики добавят новые текстовые ключи. Запустите `diff-update.bat`, чтобы найти недостающие строки. Существующие переводы продолжат работать.

**Как вернуть английский язык?**
- Уберите параметры запуска в Steam (или просто запустите игру обычным способом). Можете также удалить `code\ru-locale.jar`.

**Другие игроки видят мой русский текст?**
- Нет. Перевод работает только на вашем компьютере. Другие игроки видят игру на своём языке.

**Работает ли с не-Steam версией?**
- Скрипт запуска рассчитан на Steam-версию, но при адаптации classpath может работать и с другими.
